## 📌 About This Project

**Lanhosa** is a **heartfelt, interactive personal website** built with HTML, CSS, JavaScript, and Gulp.

This project was created as a special digital gift — a beautiful, animated experience to express love, memories, and meaningful moments through code. Every detail is crafted with care, because some feelings deserve to be shared in the most creative way possible.

🌐 **Live Demo**: [https://domlovesaforever.vercel.app](https://domlovesaforever.vercel.app)

---

## 🚀 Key Features

* 💌 **Personalized Message** – A custom love letter with heartfelt content
* 🎨 **Romantic UI Design** – Soft colors, elegant typography & warm aesthetics
* ✨ **Smooth Animations** – CSS transitions & JavaScript interactions for emotional impact
* 📱 **Fully Responsive** – Looks beautiful on mobile, tablet & desktop

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/SereyodamChek/lanhosa.git
cd lanhosa
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server with Gulp

```bash
# Start Gulp watch & local server
gulp
# or
npm run dev  # if configured in package.json
```

Then visit: [http://localhost:3000](http://localhost:3000) *(default Gulp port)*

---

## 📦 Build for Production

```bash
# Run Gulp build task
gulp build
# or
npm run build
```

Optimized files will be output to the `/assets` or `/dist` folder.
